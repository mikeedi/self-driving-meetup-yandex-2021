# Задача на селф драйвинг митап от яндекса 2021  
Условие https://contest.yandex.ru/contest/28643/enter/   
Эвристика - использовать только одного робота и брать только тот заказ, который после выполнения (с учетом времени до места заказа) даст максимальную прибыл.  
Решение работает только для 5/10 тестов, оно не самое эффективное (но и задача не самая интересная).
